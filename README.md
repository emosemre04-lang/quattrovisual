# Quattro Visual

Site vitrine officiel de Quattro Visual — agence de création de sites web.

## Structure

- `index.html` — Page d'accueil
- `mentions-legales.html` · `cgv.html` · `confidentialite.html` · `cookies.html` · `cgu.html` — Pages légales
- `legal.css` — Styles partagés des pages légales
- `assets/` — Logo + captures portfolio

## Déploiement

1. Connecter le repo GitHub à Vercel

3. Build Command : *(vide)*
4. Output Directory : `./`

Activer dans Settings → Pages → Branch `main` / root.
